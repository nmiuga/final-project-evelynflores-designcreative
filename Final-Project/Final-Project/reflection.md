# Reflection

## Overall Learnings
I learned that it is okay to seize opporunities that may create a different vision from one initialy had. I describe this more in the challenges, but I ultimatley let go of trying to control every single detail and design, and in doing so, it created a good layout, which ended up looking like how I wanted it to. I also really came to appreciate GitHub. Every time I made a significant change, I committed a version of the code and labeled what I had done. Because of this, I was able to go back to this version of the app because I was trying to add more features, but it ruined the design and functionality instead.

## Challenges
Since I had designed this app on Figma before, I was trying to describe every detail of my app in my initial prompt, but it kept getting it wrong. I tried the initial prompt multiple times, but since it was in the same conversation, I think it was pulling things from other prompts I had typed out, even though I had asked it not to. Ultimately, I decided that I would let it control how it formats things. I gave it a brief description of what I wanted, but I wasn’t thinking about every single detail anymore. I thought this would be an opportunity to see how else this app could look. After that, I gave it other edits/suggestions that just complemented the initial design.

## Next Steps
Because of time, I decided that this was a good stopping point. I would love to fully integrate an animated splash screen, which I know with more time I could’ve done. I also started to add more features, such as editing and adding new assignments, but in doing so, it changed the layout and functionality of the app, so I retracted to this version.

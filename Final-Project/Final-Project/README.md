# Native App Development — Final Project - "productiBEEty"

Starter repository for NMIX 4030/6030. See the assignment page for full instructions.

## What to submit

Your repo should contain the following when you submit:

- Your Xcode project folder (the complete app)
- - `prompt.md` — your initial prompt (replace the placeholder)
  - - `edits.md` — numbered list of manual adjustments (replace the placeholder)
    - - `reflection.md` — 150-300 word reflection (replace the placeholder)

 a polished project README visible to anyone who visits your GitHub repo. It should include your app name and a short description, at least 2–3 screenshots of the running app, and brief setup instructions. Store screenshots in an assets/ folder at the root of your repo and reference them using relative Markdown paths (e.g., ![Home screen](assets/screenshot1.png)). A starter template is provided in the repo.
